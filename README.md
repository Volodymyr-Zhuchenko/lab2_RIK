# Zhuchenko-Volodymyr-UID_lab2

Десктопний застосунок **«Калькулятор кредита та депозита»**, розроблений для лабораторної роботи №2  
з курсу **«Розробка інтерфейсів користувача»**.

---

## Опис проєкту

Застосунок реалізує кросплатформний інтерфейс для розрахунку фінансових показників.

Програма надає користувачу можливість:
- обирати тип розрахунку: **Кредит** або **Депозит**
- динамічно перемикати **світлу та темну теми** оформлення (Indigo Modern)
- вводити дані з автоматичною валідацією та форматуванням чисел
- отримувати результат розрахунку у текстовому полі
- зберігати результати у **локальний текстовий файл**

---

## Технологічний стек

**Платформа**
- Node.js
- Electron.js (Runtime)

**Технології інтерфейсу**
- HTML5 (Розмітка)
- CSS3 (Flexbox, CSS Variables для тем)
- JavaScript (ES6+, Vanilla)

**Стилізація**
- Кастомні радіокнопки у стилі Indigo
- Адаптивні SVG та локальні PNG іконки (Sun/Moon)
- Шрифт: Segoe UI

**Збереження даних**
- Модуль Node.js **fs** (FileSystem)
- Кодування: UTF-8

---

## Структура проєкту

**finance-electron/**
- `main.js` — головний процес (створення та керування вікном)
- `renderer.js` — логіка розрахунків, валідація та обробка подій
- `index.html` — структура користувацького інтерфейсу
- `style.css` — оформлення, стилі тем та компонентів
- `images/` — папка з локальними іконками (sun.png, moon.png)
- `.gitignore` — конфігурація виключень для Git (node_modules/)
